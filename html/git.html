Git

  • git init- инициализирует новый (или существующий) репозиторий Git в текущем каталоге. Это создает скрытую папку .git, которая отслеживает всю историю проекта.
  • git add - Перемещает изменения из рабочего каталога в "индекс" (или "stage"). Это подготовка файлов к следующему коммиту.
  • git commit - изменения, которые были добавлены в индекс с помощью git add. Это создает "снимок" состояния проекта на данный момент в истории.
